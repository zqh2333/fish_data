## 修改 APK 教程

### 1️⃣ Fork 本仓库

在 GitHub 页面点击 **Fork** 按钮，将本仓库复制到你的账户下。

---

### 2️⃣ 下载 APK

下载文件：
`1.4.0_7.15_sub.apk`

---

### 3️⃣ 打开 APK

使用任意反编译工具（如 **MT 管理器**）打开 APK。

---

### 4️⃣ 修改内容

1. 打开 `classes.dex` 文件
2. 全局搜索以下字符串：

https://fish.enlysure.com


3. 将其替换为：

你的仓库地址/raw/master


✅ 示例：
https://github.com/你的用户名/你的仓库名/raw/master


---

### 5️⃣ 回编译并签名

完成替换后，重新编译 APK 并进行签名。
签名完成后即可正常使用。
